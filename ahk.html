You can use AHK to farm stuff in stupid grindy flash games, like me no-lifing here.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

AHK (Auto Hotkey) is free, and you can make some pretty nice macros with it. With that over, here are the basic commands of this Not So Hard to understand program:

 

;    <- if you put that semi colon anywhere in your code, the program doesn't read it when you run it.

 

MouseMove, X, Y      <- There should be a program named "Active Window Info" that automatically gets installed when you install AHK. To get the Coordinates of

                                       where you want to move your mouse, simply click the place you want the mouse to go and choose the "relative" one.

 

Sleep, X     <- a delay. 1000 = 1 second. So if you wanted your program to wait 4 seconds before clicking or something, then: Sleep, 4000

 

click X Y     < note the no capitals or commas with this one. If you already decided to move your mouse, then there no need for the coordinates, just input "Click" onto

                     the line, w/ a capital.

 

LabelName:      < If you wanted to repeat something, then put this at the front of your code.

Goto, Labelname     < Compliments the previous command

 

Pause On      < This command, if you put it at the very front of the command it will make is so that the program doesn't immediately run when you open the macro up.

 

Send a     < sends a letter

 

 

__________________________________________________________________________________________________________________________________

Here is an example of what I was talking about

 

; Main Script   (labels the thing you are working on)

; ----------------------------

Pause On    ;show that program doesn't instantly go

 

 

MyLabel:

 

MouseMove, 280, 715          ; Simply moves the mouse

Sleep, 200                 ; Waits for a bit and then moves it some more

click                ; Clicks once

 

MouseMove, 920, 715       ; same thing as above

Sleep, 200

click

 

Goto, MyLabel    ;goes back to the "MyLabel" so the program loops

 

^s::Pause         ;This command starts the Macro

^d::ExitApp        ;This command Exits the App for me
